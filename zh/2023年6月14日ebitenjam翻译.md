# Ebiten Game Jam 2023

```编辑（2023年6月14日）：我们已经纠正了Game Jam的开始时间，之前设置错误了2个小时。对此给您带来的任何不便，我们深感抱歉。```

Ebitengine Game Jam是由[Ebitengine](https://ebitengine.org/)社区组织的为期两周的活动，将于6月15日开始，任何人都可以通过构建基于秘密主题的游戏来展示您的Ebitengine游戏。

秘密主题将在2023年6月15日17:07:14 (UTC+9:00,即北京时间当日16:07:14)（Ebitengine初始提交的时间戳）公布😉，这也是您开始制作游戏的时间，您可以在接下来的两周内随时提交作品。

目前，主题建议已经关闭，主题将在活动开始时公布！

对Ebitengine还不熟悉吗？加入我们的[Discord在线社区](https://discord.gg/3tVdM5H8cC)吧，这里有活跃的讨论，包括关于本次Jam的讨论，所以带上您的话题、问题和创意，加入进来吧！

这是一个社区游戏Jam，[请遵守Go社区行为准则](https://go.dev/conduct)。

您可以在这里看到去年Jam的参赛作品：[Ebitengine Game Jam 2022: Results](https://itch.io/jam/ebiten-game-jam/results)，还有一些[使用Ebitengine制作的其他游戏](https://itch.io/c/2030581/made-with-ebitengine)。

## Jam 规则 
- 您的作品必须使用Ebitengine编写 
- 必须与主题相关 
- 游戏的代码必须是开源的，以便其他人可以学习

### 详情 
问：我可以组队参加，还是必须一个人完成？

答：允许团队参赛，团队可以有任意大小。如果您还没有队友，可以在Discord上询问是否有人愿意加入您的团队。

问：我必须组队才能参加吗？

答：不，您也可以一个人完成，很多人都会这样做。

问：我可以提交之前制作的游戏吗？

答：不可以，为了保持公平，您必须在提交期间（6月15日至29日）制作游戏。

问：我可以使用现有的资源吗？

答：可以，您可以使用预先制作的资源，但我们鼓励您制作自己的资源。制作游戏的艺术和声音是创作过程中令人兴奋的一部分，但我们不希望这成为参加Jam的障碍，因此使用适当许可的资源是可以的！

问：我可以使用Ebitengine以外的外部库吗？

答：可以，一个好的起点是[awesome-ebiten](https://github.com/sedyh/awesome-ebiten)列表，可以在那里找到与Ebitengine配合良好的有用库。

问：如果我以前从未使用过Go或Ebitengine怎么办？

答：太好了，这是尝试它们的绝佳机会，欢迎首次尝试！第一步：加入我们的Discord社区，打个招呼，那是获取帮助、支持和灵感的最佳地方，特别是当您刚开始时。如果您已经了解其他编程语言，可以通过Go之旅来熟悉Go，并且可以在Ebitengine示例页面上看到一些示例代码。如果您是编程新手，Go将是您的第一门语言，那可能会很有挑战性，但并非不可能，尤其是如果您提前练习一下，所以一定要尝试一下！

问：如果我有其他问题没有在这里回答怎么办？

答：请加入我们的Discord聊天，在#game-jam频道中提问，我们非常乐意解答！

## 评分标准
 提交期结束后（6月29日），我们将有两周的时间来玩彼此的游戏并根据以下标准对其进行评分：

- 主题解释 
- 图形 → 游戏的视觉设计 
- 音频 → 包括音乐和音效 
- 游戏性 → 游戏机制的趣味性 
- 创新 → 😉 

在评分期结束后（7月13日），将宣布每个标准的获胜者，以及总体获胜者。不要忘记，制作和玩游戏比赢得比赛更重要。
